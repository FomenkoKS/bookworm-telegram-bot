# Bot for reading club
## using:
- docker-compose
- poetry
- loguru
- aiogram

## Preparation
```sh
mkdir bot && cd $_
git clone https://github.com/FomenkoKS/bookworm-telegram-bot.git .
cd bot
```
Change .env past your TOKEN from BotFather (save and quit)


## Starting
```sh
docker-compose up -d
```
